## Instructions
### Install
- Clone repository
```sh
# https
git clone https://github.com/Gioia3022/DevOps-Project-G-A.git
```
```sh
# ssh
git clone git@github.com:Gioia3022/DevOps-Project-G-A.git
```
```sh
# github CLI
gh repo clone Gioia3022/DevOps-Project-G-A
```
- Install redis and npm
