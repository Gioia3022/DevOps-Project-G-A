# Vagrant

## Usage

#### Access the `iac/` folder

Clone the current Git repository.

#### Establish a virtual machine

This is possible by executing the following command:

```sh
vagrant up
```

#### Open the URL in your browser

http://192.168.56.2:3000/

#### 4.4 Shutdown the VM

```sh
vagrant halt
```

#### Delete th VM

```sh
vagrant destroy
```
